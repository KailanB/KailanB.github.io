Favorite Selectors

Selector Written Description
element h1 Selects all h1 elements
.class .examples Selects all elements with class="examples"
#id #example Selects the element (id's must be unique) with id="example"
* * Selects ALL elements
element, element p, h1 Selects all p and all h1 elements
element element div p Selects all p elements within div elements
element.class p.examples Selects all p elements with class="examples"
element>element div>p Selects all p elements with the parent element div
element+element div+p Selects the all p element that are directly after div elements
element::first-child ::p Selects all p elements that are the first child
element:first-of-type p:first-of-type Selects all p elements that are the first p of its parent
element:last-child p:last-child Selects all p elements that are the last child of its parent
element:last-of-type p:last-of-type Selects all p elements that are the last p of its parent
:nth-child(n) p:nth-child(n) Selects all p elements that are the nth child of their parent
:nth-of-type(n) p:nth-of-type(n) Selects all p elements that are the nth p of their parent

Favorite Properties and Functions

Property or Function Example Description
flex display: flex; Sets a container to function using the flex-box layout. This allows items such as paragraphs or other containers within the flex-box to easily adapt to different screen sizes
flex-direction flex-direction: column; / row / row-reverse / column-reverse Changes the direction that flex items within the box flow. Row runs left to right, column top to bottom
flex-wrap flex-wrap: nowrap; / wrap / wrap-reverse Changes the behavior of flex-items based on screen size. When items are set to wrap they start a new row or column when they run out of space. Default is nowrap
box-sizing box-sizing: content-box; / border-box Adjust whether or not a container takes the padding and border into consideration of it's total width and height. Border-box includes padding and border in total width and height. Default is content-box.
text-decoration text-decoration: none; / underline / line-through Adjsuts text to have decorations or not. Can also be used to remove dots on lists, or underlines on urls.
min() min(40vw, 200px) Chooses the lesser of the two input values as the width. Can also use max() to choose larger of two values
calc() calc(10-5) Performs a calculation and uses the result as the value. Can be used with variabes or percentages
rgb() rgb(55 20 80); Changes the colour of something based on the RGB scale
