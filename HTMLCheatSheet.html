Structure

 Tag  Description
HTML Used to create the entire website
head Contains metadata for the entire website that is not displayed
body Creates the visible part of the website
title This is the name of the title bar, and is the name that will be bookmarked

Basics

 Tag  Description
h1 - h6 Page headings, with size and significance starting with 1 as largest/highest
p Basic paragraph
strong Bold text
em Italic text
br Creates a line break
hr Horizontal line
a Hyperlink
div Creates an area to put data within
ol Ordered list
ul Unordered list
list List item within an ol or ul

Tables and Forms

 Tag  Description
table Creates a table
tr Table row within a table
th Table header, text is bold
td Table data cell
form Creates a form used to collect data
label Creates a label for input boxes
input Creates an input method, written text, check boxes etc.
textarea Creates a box for a user to write for comments etc.
button Creates a button for submitting data
fieldset Creates a border for areas of a form
legend Creates a title for a fieldset
